## 梦境

```
梦境的场景来自于:小学的学校位于一座稍微高的山坡顶上, 我的家离学校很远,但也不是遥不可及.

在一片漆黑的夜晚,伸手不见五指,似乎在这个梦境世界永远都是这样的黑夜. 我虽然看不清脚下的路,也感受不到身边任何的事物,只有空旷, 但我很清楚我的目标是位于小学学校右上角的”垃圾堆“上,而小学学校就在一座稍微高个几十米左右的山坡顶上,因为那儿有一火焰光团正在散发着模糊但持续的光,指引着身在远方的我.

在这样黑暗的环境中,我虽然看不见身边的事物,但摸着脚下的道路,类似在呈现“z”字形的道路上一直走着. 为什么会有道路呢? 或许当我看见或者感受到那火焰光团时,就已经形成了道路了吧.

在一直朝着那火焰光团靠近的过程中,它一直都在那儿,夜晚中持续散发光芒,不强烈,甚至有些模糊,但我非常清晰的感受它的位置.  当我走到一个山坡的山脚时,我发现位于山坡顶的右上角落就是我的一直感受到的那火焰光团所在位置. 而在山坡顶的左下角则有另外一团稍小的火焰光团.这团稍小的火焰光团的位置类似于小学学校的大门处. 而此刻我知道:要想去那团我心中一直存在的那火焰光团的位置,就要先路过这团稍小的,或者说要先进入这个学校打大门.

在我发现这团稍小的火焰光团后,也明确我当下的目标,道路,计划后. 我开着“拖拉机”,一路弯弯扭扭的到了大门处,在这段路上,虽然看不见身边的障碍物,但我就是知道我每次弯弯扭扭绕开的地方就是障碍物,确切的说是我通过“开拖拉机”这种方式必须要绕开的”障碍物“. 而这个拖拉机在现实中就是我父亲的拖拉机.

当我进入学校大门后,也进入了那团稍小的火焰光团后,我似乎进入到了一个类似操场大小空间的屋子内.屋子里面很明亮,像没有任何阴影的那种明亮,那里有“老人”,小孩,男人,女人,但都是我所熟悉的亚洲人面孔. 也有几个女人似乎好奇的看到了我,我先向她们打招呼说“你好”,她们也一边好奇一边回应说“你好”. 然后我看到身边有着非常大的,连排的冰柜, 冰柜里放的都是一包包已经用密封袋装好的食物,每包食物都对应着一个人的名字, 这些名字,似乎是这个世间所有人的名字,也包括在这个屋子里面的所有人的名字. 每个袋子里面的食物有的是羊肉,有的是荤素搭配好的,有的是海鲜和蔬菜混合的. 每个袋子的大小看起来是差不多大的,重量也差不多. 

这时候我看到一个“普通的中年妇女”来拿属于她的那袋食物,这个女人似乎说她的食物不够,她还想再拿,于是她伸出她“黑乎乎”的肥手直接将冰柜里写着其他人名字的袋子打开,从中直接抓出了一半的海鲜和蔬菜. 旁边似乎还有一个冰柜管理员在,冰柜管理员看到这种情况了,似乎不忍,但没有阻止,也没出声,默认了. 但为什么呢? 应该是这个“普通的中年妇女”是属于”这个屋子内“的人吧.  当我看到这个场景后,我充满了鄙夷. 那个冰柜管理员告诉我:你进入到这个屋子里面了,除了你自己那袋食物,可以随意再拿取“一定量”的写有其他人名字的袋子里面的食物.  但我拒绝了,此时梦境场景转移了.

同样是一个非常明亮的空间,没有任何阴影. 我走向一个超大的木桌前, 这个木桌可以坐上一百人, 但此时看起来似乎一半的座位都没满.这时候有个中年男人带着一个他刚成年的儿子正在坐在木桌的座位上, 这个中年男人向我轻微的点头示意友好后,就没有再看向我了. 而他那个刚成年的儿子则是一脸阴险的直盯着我,似乎在打着不好的心思. 我刚看到这个场景后,没有理会那个阴险的年轻人, 而是友好的和木桌旁那些人友好的轻聊. 从听到他们聊天后发现几乎这些人身上都有一条有着特定的大量人脉和资源纠缠的关系链,每个人都不可忽视,所以每个人都会友好的交谈.  此时或许是我感受到那个阴险的年轻人盯着我的时间太长了,我愤怒的怼了他一句“小逼崽子,你看谁呢!”. 而那个阴险的年轻人轻蔑的笑了一下,转过头看了一下他的父亲,他的父亲也就是那个中年男人回头看了我,从一开始的优雅绅士友好的笑容 转为了一脸阴沉的脸看着我. 我自认为自己没有做错什么,没有屈服,直接对视他的眼睛. 此时身边的其他人似乎发现了什么, 一开始4,5个人过来看着我,我没有屈服. 然后是20个人过来围着我,就那么静静地看着我,我也没有屈服. 我此时感觉到了什么叫“群狼环绕”的情况了. 压力一下子大了起来, 但仍然在我承受的压力极限之内. 下一刻,梦醒了.


```

## 自我反思

```
1, 当看到有“脏手”去抓取不属于自己的资源时,不要鄙夷,也不要自视清高,这是现实的一种现象.

2, 祸从口出, 沉的住气,多听,多看,多观察,多思考,少说话. 脸部的表情不要暴露自己的情绪和心声.

3, 构建属于自己的人脉和资源的关系网.

4, 冷静,低调,隐藏,学习.

```
